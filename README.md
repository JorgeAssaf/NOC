## Installation

1. Clone the repository

```bash
  https://github.com/JorgeAssaf/NOC.git
```

2. Go to the project directory

```bash
  cd NOC
```

3. Up the docker-compose

```bash
  docker-compose up -d
```

4. Install the dependencies

```bash
  npm install
```

```bash
  pnpm install
```

5. Run the project

```bash
  npm run dev
```

```bash
  pnpm dev
```
